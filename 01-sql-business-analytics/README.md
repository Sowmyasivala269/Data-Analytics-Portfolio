# SQL Business Analytics

## Project Overview
Analyzing real business data using advanced SQL techniques to answer
key business questions and surface actionable insights.

## Business Questions I'm Answering
- What are the top performing products by revenue?
- Which customers are at risk of churning?
- What is the month-over-month revenue growth?
- Who are the top salespeople in each region?

## Tools Used
- PostgreSQL
- Python (Pandas, Matplotlib)
- Jupyter Notebook

## Skills Demonstrated
- Window functions (ROW_NUMBER, RANK, LAG, LEAD)
- CTEs (Common Table Expressions)
- Subqueries
- Date functions
- Query optimization

# Social Media Engagement Analytics

## Project Overview
Analyzed 500+ social media posts to identify what content 
drives the highest engagement and provide actionable recommendations.

## Problem Statement
A content team needed data-driven guidance on which content 
categories to prioritize to maximize audience engagement.

## Approach
1. Cleaned dataset — removed nulls, duplicates, fixed data types
2. Visualized engagement trends using histograms and boxplots
3. Performed statistical analysis across content categories
4. Identified top performing content types

## Key Findings
- Culture & travel content generated 25% higher engagement
- Recommended content mix changes projected to raise average likes by 18%
- Identified optimal posting patterns by category

## Tools & Technologies

- Pandas (data cleaning and analysis)
- Seaborn (visualization)
- Jupyter Notebook
